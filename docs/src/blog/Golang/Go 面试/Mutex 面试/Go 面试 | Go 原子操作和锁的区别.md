| 特性         | 原子操作                                               | 锁                                             |
| ------------ | ------------------------------------------------------ | ---------------------------------------------- |
| **底层实现** | 由底层硬件直接支持                                     | 基于原子操作和信号量实现                       |
| **性能**     | 通常更高效，因为它是单个指令的执行                     | 相对较低，涉及到上下文切换                     |
| **适用场景** | 单个指令的互斥操作                                     | 处理临界区（多个指令）的互斥操作，扩大操作范围 |
| **锁类型**   | 无锁操作，属于乐观锁                                   | 通常属于悲观锁                                 |
| **层级存在** | 存在于机器指令层、汇编指令层、高级语言层（如 Go 语言） | 同样存在于不同指令和语言层级                   |
